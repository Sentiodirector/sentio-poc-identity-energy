# Classroom Engagement Heatmap & Group Analysis

Turn individual face detections into a room-level engagement picture with spatial zones.

## Problem
When we track individuals but schools want to understand the room as a whole: which 1 minute windows had peak attention? Which seats are chronically disengaged?

## Input
- `video_sample_1.mov` (fixed overhead or front-facing classroom camera)

## Output
1. `engagement_report.html` - Timeline chart + spatial heatmap
2. `engagement_output.json` - Integration data for Sentio Mind
